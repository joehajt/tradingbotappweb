# Trading Bot Web Application

Automated cryptocurrency trading bot with Telegram integration and Bybit API support.

## Quick Start

### 1. Create Virtual Environment
```bash
python3 -m venv venv
```

### 2. Activate Virtual Environment
```bash
source venv/bin/activate
```

### 3. Install Dependencies
```bash
pip install -r requirements.txt
```

### 4. Run the App
```bash
python app.py
```

### 5. Open in Browser
http://localhost:5001
